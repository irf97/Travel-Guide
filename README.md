# Social Travel Intelligence Dashboard

Investor-facing static dashboard for comparing young-adult Mediterranean travel destinations by budget, social density, nightlife, historic depth, nationality mix, gender-ratio proxy, and month-by-month seasonality.

## Files

- `index.html` — the full single-file dashboard.
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing.

## Hosting with GitHub Pages

1. Go to this repository on GitHub.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch: `main`.
5. Select folder: `/root`.
6. Save.

The live site should become available at:

```text
https://irf97.github.io/Travel-Guide/
```

Note: if the repository remains private, GitHub Pages availability depends on your GitHub plan. For free public hosting, make the repository public.
